# Battle Field 11A 黒薔薇領

- BF10でロザリィ撃破した場合のルート
- 2部構成
- 出撃枠3人
- 友好軍ロズウェル
- ロズウェルに隣接で11A-2へ（カード復活）

## 勝利条件 

11A-1
- 自軍ユニットがロズウェルに隣接

11A-2
- ロズウェルが目標地点(G02)に到達

## 戦闘中イベント 

11A
- TNV10以上で集落に行くと「魔水晶」→「クリスタルロッド」

11A-1

11A-2
